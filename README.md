# Library Database Management System

This repository contains SQL scripts for an academic project developed as part of the
Advanced Database (IT332) course in the Information Technology program.

## Project Overview
The system is designed to support core library services, including:
- Catalog management (Books, Periodicals, Theses, Media)
- Member management
- Loans and fines
- Room reservations
- Digital resources

## Features
- EER-based conceptual database design
- Relational schema design and normalization (1NF–3NF)
- SQL implementation (DDL & DML)
- Views and role-based access control

## Technologies
- SQL
- Oracle Database

## Repository Contents
- `createDB_1.sql` – Database schema and table creation
- `create_DataAccess_1.sql` – Roles and access control
- `createvBookStatus1.sql` – Database views
- `test_View_1.sql` – Sample test queries

## Notes
This project is intended for educational and demonstration purposes only.
